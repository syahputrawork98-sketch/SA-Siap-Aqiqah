# Dokumentasi Seed Data Per Tabel

Folder ini berisi detail data awal (seed) yang dimasukkan ke dalam setiap tabel database Siqah. Pemisahan dokumen ini bertujuan untuk mempermudah kontrol data master selama fase pengembangan.

## Daftar Tabel Seed

1. **[Platform Settings](platform-settings.md)**
   Konfigurasi global seperti DP dan fee platform.
2. **[Users](users.md)**
   Data akun autentikasi untuk seluruh persona pengembang.
3. **[Consumer Profiles](consumer-profiles.md)**
   Profil detail untuk akun konsumen.
4. **[Partner Profiles](partner-profiles.md)**
   Profil detail untuk mitra (Kandang, Catering, Kurir).
5. **[Packages](packages.md)**
   Katalog paket aqiqah utama.
6. **[Animals](animals.md)**
   Katalog hewan yang tersedia di mitra kandang.
7. **[Catering Menus](catering-menus.md)**
   Katalog menu masakan yang disediakan mitra catering.

---
*Gunakan dokumen ini sebagai referensi saat melakukan pengujian fitur yang bergantung pada data master.*
